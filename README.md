# QIIME 2中文文档(QIIME 2 Chinese Manual)

本项目是QIIME 2官网的中文翻译：

- 英文原版每季度更新，目前更新至2021.2，下次更新2021.4
- 中文简明教程，每季度更新一次，与英文原版同步
- 中文完整文档，每季/半年更新一次

软件更新较快，如果有问题以英文官网教程为准
